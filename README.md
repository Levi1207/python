## 1. Общее описание проекта

**Цель:** Создать систему распознавания жестов рук в реальном времени с помощью веб-камеры ноутбука, используя готовый датасет для обучения.

**Задачи:**
- Загрузить и проанализировать датасет жестов
- Построить сверточную нейросеть (CNN) для классификации
- Обучить модель различать 5-10 жестов
- Реализовать демонстрацию с веб-камерой
- Оценить точность и скорость работы

**Жесты (на примере HaGRID):**
| № | Жест | Название | Условное обозначение |
|---|------|----------|---------------------|
| 1 | ✋ | Stop/Palm | `stop` |
| 2 | ✊ | Fist | `fist` |
| 3 | ✌️ | Victory | `victory` |
| 4 | 👍 | Thumbs up | `thumbs_up` |
| 5 | 👌 | OK | `ok` |
